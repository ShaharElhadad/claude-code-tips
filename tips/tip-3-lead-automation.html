מה Claude Code בונה לכם

מערכת מענה אוטומטי שלמה, בפרומפט אחד

אתם מתארים מה אתם רוצים, Claude Code בונה את הכל. ממש הכל.

מה נבנה

סקריפט שמאזין לטפסים ועונה מיד

Claude Code ייצור לכם סקריפט Node.js שמתחבר לטופס שלכם (Google Forms, Typeform, טופס באתר), ומרגע שליד ממלא פרטים, שולח לו מייל מותאם אישית תוך 60 שניות.

המייל לא גנרי. הוא מתייחס לשם הליד, לשירות שביקש, ומוסיף שורה שמזמינה לשיחה.

הכל אוטומטי

אתם לא נוגעים בקוד, Claude Code עושה הכל

אתם לא צריכים לדעת JavaScript, לא צריכים להבין APIs, לא צריכים לחבר שום דבר ידנית.

אתם פשוט אומרים ל Claude Code מה אתם רוצים, והוא: כותב את הקוד, יוצר את הקבצים, מגדיר את החיבורים, ואפילו בודק שהכל עובד.

התוצאה: כל ליד שנכנס מקבל תשובה מותאמת אישית תוך דקה, גם אם אתם ישנים, באמצע פגישה, או בחופש.
העתיקו והדביקו

3 פרומפטים שעושים את העבודה

פתחו Claude Code, העתיקו את הפרומפטים האלה, ותנו לו לרוץ. תוך 3 דקות הכל עובד.

1

הפרומפט הראשון: יצירת הפרויקט

מתחילים מתיקייה ריקה. העתיקו את זה ל Claude Code:

Claude Code
# Prompt 1: Create the project
Create a Node.js project called "lead-responder"
that does the following:
1. Runs an Express server on port 3000
2. Has a POST endpoint /webhook that receives
lead data (name, email, phone, service)
3. When a lead comes in, sends a personalized
email using Nodemailer with Gmail SMTP
4. The email should greet the lead by name,
mention the service they asked about,
and include a link to book a call
5. Log every lead to a local JSON file
6. Use .env for Gmail credentials
7. Add error handling and retry logic
Set up everything: package.json, .env.example,
.gitignore, and a README with setup instructions.
Install all dependencies.
2

הפרומפט השני: התאמה אישית של המייל

אחרי שהפרויקט נוצר, תגידו ל Claude Code איך אתם רוצים שהמייל ייראה:

Claude Code
# Prompt 2: Customize the email
Update the email template:
Subject: "היי {name}, קיבלנו את הפנייה שלך"
Body should be in Hebrew, warm and personal:
- Greet by first name
- Mention the specific service they asked about
- Say we'll be in touch within 2 hours
- Add a Calendly link to book a call
- Sign with my name: [YOUR NAME]
Make the email HTML with clean design,
mobile responsive, RTL support.
No images, just clean typography.
3

הפרומפט השלישי: חיבור לטופס ובדיקה

עכשיו מחברים את הטופס ובודקים שהכל עובד:

Claude Code
# Prompt 3: Connect and test
Now do the following:
1. Create a test script that simulates
a lead submission to /webhook
with sample data
2. Add a simple HTML form at /form
that I can embed on my website or
share as a standalone link
3. Add a /leads endpoint that shows
all leads received so far (protected
with a simple API key from .env)
4. Run the test and confirm the email
was sent successfully
שימו לב: החליפו את [YOUR NAME] בשם שלכם, ואת קישור ה Calendly בקישור שלכם. הכל השאר Claude Code עושה לבד.
אחרי שזה רץ

מה לעשות עם זה עכשיו?

1

חברו את הטופס לאתר שלכם

הטופס שנוצר אפשר להטמיע באתר עם iframe, או פשוט לשלוח את הקישור ישירות ללקוחות.

2

הריצו את השרת על מכונה קבועה

בקשו מ Claude Code: "Set up PM2 to run the server permanently and restart on crash". הוא יגדיר הכל.

3

הוסיפו התראה לטלפון

בקשו: "Add a Telegram notification when a new lead comes in, send the lead details to chat ID [YOUR_ID]". ככה תדעו בזמן אמת על כל ליד.

4

שדרגו עם follow up אוטומטי

בקשו: "Add automatic follow up: if no reply after 24 hours, send a second email. After 3 days, send a final one." Claude Code יבנה את כל הלוגיקה.

שאלות נפוצות

שאלות ותשובות

אני לא מתכנת, זה באמת עובד בשבילי?
כן. כל הפרומפטים כתובים ככה ש Claude Code עושה הכל. אתם לא צריכים לגעת בקוד. אם משהו לא עובד, אתם פשוט אומרים לו "this doesn't work, fix it" והוא מתקן. הוא גם מסביר לכם מה הוא עשה, אז אתם לומדים תוך כדי.
למה לא פשוט להשתמש ב Zapier או Make?
Zapier עולה $20 עד $100 בחודש ומוגבל ב tasks. כאן אתם בונים משהו שנמצא על השרת שלכם, ללא הגבלת שימוש, וללא תשלום חודשי. בנוסף, עם Claude Code אתם יכולים לשדרג ולהתאים כל דבר. תנסו לעשות את זה עם Zapier.
איפה אני מריץ את זה?
על כל מחשב שרץ 24/7. מחשב ישן בבית, VPS ב 5 דולר בחודש (DigitalOcean, Hetzner), או Raspberry Pi. Claude Code יכול גם להגדיר לכם deploy אוטומטי. פשוט תגידו לו: "deploy this to my server at [IP]".
אפשר להוסיף WhatsApp במקום מייל?
בהחלט. אחרי שהמייל עובד, תגידו ל Claude Code: "Add WhatsApp notification using the WhatsApp Business API. Send the same personalized message via WhatsApp too." הוא יוסיף את זה על הקיים, בלי לשבור שום דבר.
